About HTIF
******************

**HTIF (High throughput interaction filter)** is a lightweight package for evaluating docking poses based on their interactions with a target protein.
Built for speed and low memory overhead, it scales easily to large virtual screening datasets, making it well suited for high-throughput analysis.

HTIF can be used to:

- Filter docking poses according to user-defined interaction criteria
- Characterize the full interaction profile between a ligand and its target protein


Check out the :doc:`usage` section for further information, including
how to :doc:`installation` the project.
For more information about the interactions used in HTIF, please refer to the :doc:`interactions` section.

.. note::

   This project is under active development.
   For any feedback, please refer to the :doc:`feedback` section.
